递归

· 2019-06-05 · # JavaScript

递归是一种编程模式，当一个函数调用自身时，那么它称为递归；
//我们写一个函数，计算a的b次方，用a乘以自身b次
//递归方法

function pow(a, b) {    
  if(b == 1) return a;  
  return a *= pow(a, b-1);  
}

//循环

function poe(a ,b) {  
  let res = 1;  
  for(let i = 0; i < b; i++) {  
    res *= a;  
  }  
  return res;  
}

最大的嵌套调用次数包括首次，称之为递归深度
任何递归都可以用循环来重写。循环变体一般更加有效。
